# Data Analysis Techniques

## 1. Course Objectives

The objective of this course is to introduce students to data analysis using the Python programming language.
The course will cover the most popular libraries for data processing and visualization.

## 2. Sylabus

[Data+Analysis+Techniques+2026L.pdf](Data+Analysis+Techniques+2026L.pdf)

**Main topics during the lecture:**

Examples during laboratories will cover code snippets with real world data.

* Working with Jupyter Notebook (standalone or via Google Colab).
* NumPy amd Pandas libraries.
* Statistics and linear algebra in Python.
* Data visualization with matplotlib, seaborn, plotly and bokeh.
* ETL process with data - Extract, Transform, Load.
* Data cleansing.
* Dimensionality reduction.


## 3. Tools

All the materials and examples will be delivered as Jupyter Notebooks, which might be run locally on student's computers, lab computers or via Google Colab. We will work with Python version 3.

## 4. Conditions for passing the course

Students are required to submit solutions to the assignments given during class by the start of the next lab session.
